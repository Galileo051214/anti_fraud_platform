# 复杂诈骗套路深度解析

> 数据来源：2025~2026年公安反诈中心通报、权威媒体反诈报道
> 用途：反诈骗学习平台 —— 案例库/资讯内容填充

## 一、杀猪盘 · 复合式情感投资诈骗（单案损失TOP1）

### 1.1 经典杀猪盘（传统四步）

| 阶段 | 名称 | 详细手法 |
|------|------|---------|
| ① | **选猪** | 通过婚恋平台（世纪佳缘、探探、Soul）、社交APP、游戏公会筛选目标。偏好 25~50 岁单身/离异、有稳定收入、情感需求强烈的人群 |
| ② | **养猪** | 打造「完美人设」—— 投行精英/跨境电商老板/部队军官，每日嘘寒问暖，分享「日常生活」，持续 1~4 个月建立情感依赖。期间进行服从性测试（发送小额红包、赠送虚拟礼物） |
| ③ | **杀猪** | 时机成熟后，无意中透露「内部投资渠道」「数字货币套利」「博彩平台漏洞」，先让受害者小额试水并成功提现（返利 500~2000 元）。待受害者完全信任后，诱导大额充值 |
| ④ | **毁尸** | 受害者要求提现时，以「账户冻结需解冻费」「流水不足需刷流水」「税金」「保证金」等连环收费，榨干最后一分钱。最后平台关闭、拉黑失联 |

**真实案例**：澳门 2025 年 7 月以来至少 18 名居民被骗，总损失超 2789 万澳门元。其中一名受害者被虚假投资平台「INST-SI」「TOKENPOCKET」吞噬数百万。

**洗钱路径**：诈骗赃款 → 河南粮商账户 → 购买数百吨小麦转卖变现 → 香港多级账户 → 虚拟货币洗白

### 1.2 杀猪盘 2025-2026 四大变种

#### 变种一：职场「伯乐」型

- **目标人群**：25~35 岁职场人，有跳槽或副业需求
- **手法**：在领英/Boss直聘伪装投行总监/HR，先分享行业干货、帮忙修改简历建立权威 → 推荐「跨境贸易对冲」副业 → 先转账 2000 元「诚意金」获取信任 → 诱导投入 10 万+
- **核心特点**：利用职场焦虑，先建立「导师」形象再引入骗局

#### 变种二：兴趣社群「同好」型

- **目标人群**：汉服、摄影、手账、剧本杀等小众圈层爱好者
- **手法**：潜伏在兴趣群内 2~3 个月，分享专业知识、赠送小礼物（绝版发簪/手工香囊）→ 提出投资「文创项目」「圈内资源」→ 发来伪造的工厂视频和「专利证书」
- **核心特点**：长期经营，利用圈层信任感

#### 变种三：AI 换脸 + 虚拟定位「同城男友」型

- **目标人群**：独居女性、离异女性
- **手法**：虚拟定位显示在同一城市 → AI 换脸技术合成视频通话（背景有地铁报站、办公室键盘声）→ 根据受害者朋友圈晒的定位设计「偶遇」→ 诱导数字货币平台投资 35 万后消失
- **技术手段**：AI 换脸伪造视频 + 预先录制的环境音 + 虚拟定位

#### 变种四：「反向杀猪」先打钱型

- **目标人群**：做微商/小生意的个体户
- **手法**：伪装大客户，先下单并多转几百元「运费补贴」→ 介绍几个真实「客户」建立信任 → 提出「酒水代理」让受害者垫资囤货 → 承诺「利润对半」。受害者抵押房产贷 50 万打过去后失联
- **核心特点**：利用「对方先付出」的心理降低警惕

## 二、AI 深度伪造诈骗（技术门槛最高）

### 2.1 AI 换脸视频诈骗

| 维度 | 细节 |
|------|------|
| **手法** | 通过社交平台获取受害者亲友的照片/视频，利用 Deepfake 技术生成实时换脸视频，伪装成亲友进行视频通话 |
| **常用场景** | 冒充子女要手术费、冒充老板要求转账、冒充公检法 |
| **真实案例** | 浙江王女士 2025 年收到「女儿」视频电话，画面中女儿满脸伤痕哭诉车祸需 15 万手术费，转账后核实发现女儿在校安然无恙 |
| **技术细节** | 骗子使用开源 DeepFaceLab/DeepLiveCam 等工具，只需 10~30 张目标人物照片即可生成可实时驱动的换脸模型 |

### 2.2 AI 语音克隆诈骗

| 维度 | 细节 |
|------|------|
| **手法** | 通过骚扰电话或社交平台语音消息采集目标声音样本（30 秒即可），使用 ElevenLabs 或讯飞开放平台等工具克隆声音 |
| **常用场景** | 模仿领导声音要求财务转账、模仿家人声音求救 |
| **真实案例** | 某公司财务收到「CEO」语音消息要求加急转账 85 万「合同保证金」，声音、语气、口头禅完全一致，未经电话核实直接转账 |

### 2.3 2026 新型「身份确认」语音取证诈骗

| 维度 | 细节 |
|------|------|
| **核心套路** | 不再索要密码/验证码，只问「请问是XX本人吗」，全程录音 |
| **后续操作** | 获取录音后，利用银行/运营商线上业务的语音核验漏洞，开通代扣业务、申请小额贷款、开通免密支付 |
| **六大细分场景** | ①伪装银行客服→语音核验开通代扣 ②伪装网贷客服→查询征信并申请借贷 ③电商理赔→绑定免密支付 ④社交好友→盗号申诉 ⑤冒充公检法→伪造笔录施压 ⑥运营商客服→开通付费套餐 |
| **防范要点** | 陌生来电绝不随意回答「是」，挂断后通过官方客服回拨核实 |

**数据支撑**：2026 年全国多地公安反诈中心发布风险预警，该类骗局案发量持续走高，覆盖中老年、职场年轻人、在校大学生各圈层。

## 三、刷单返利诈骗（发案量第一）

### 3.1 传统刷单

| 步骤 | 操作 |
|------|------|
| ① | 在短视频/社交平台发布「点赞赚钱」「日结高薪」广告 |
| ② | 引导关注抖音/快手账号，每单返 2~5 元，前 10 单正常返现 |
| ③ | 诱导下载「专属 APP」做「联单任务」—— 需连续完成 3~5 单才能提现 |
| ④ | 第 2 单起金额暴涨（100→500→3000→10000），完成后以「操作失误需补单」「账户冻结需解冻」为由要求继续转账 |

**真实案例**：深圳某职员被「短视频点赞赚钱」吸引，获利百余元后被诱骗垫资做「高级任务」，累计投入 8.7 万元后平台无法提现。

### 3.2 2025 刷单新变种：线下现金交付

- **手法**：不再线上转账，而是要求受害人提取现金，放置于指定地点（如小区垃圾箱、公园长椅下），诈骗团伙用**遥控车**转移赃款
- **目的**：规避银行风控系统的大额转账监测
- **真实案例**：北京李先生被刷单诈骗 13 万元，现金被遥控车取走

### 3.3 2025 刷单新变种：数字人民币钱包

- **手法**：诱导用户开通数字人民币钱包并绑定银行卡，以「充值返利」名义要求将资金充值到虚假平台，充值后无法提现
- **特点**：利用数字人民币的匿名性和便捷性实施诈骗

## 四、冒充客服系列（发案增速最快）

### 4.1 标准流程

| 步骤 | 操作 |
|------|------|
| ① | 非法获取用户网购订单信息（商品名称、价格、收货地址、订单号） |
| ② | 冒充平台/快递客服，准确说出订单信息建立信任 |
| ③ | 话术一：商品质量问题，退款 X 倍赔偿；话术二：快递丢失，双倍赔付；话术三：误开通会员/代理商，需取消否则每月扣费 |
| ④ | 诱导下载「会议 APP」（腾讯会议/Zoom/云会议等，开启**屏幕共享**功能） |
| ⑤ | 通过屏幕共享窃取银行卡号、密码、验证码，实时转走资金 |

### 4.2 「共享屏幕」技术细节

- 安卓/iOS 系统自带屏幕录制 + 远程协助权限
- 骗子通过会议软件的「共享屏幕」「远程控制」功能，实时查看受害者手机上的短信验证码
- 受害者往往以为只是在配合「客服指导操作」，实则全程被监控

**真实案例**：重庆王某接到「电商客服」电话，称其购买的商品有质量问题可三倍退款，被诱导下载「云会议」APP 开启屏幕共享，银行卡内 17.6 万元被分批转走。

## 五、冒充公检法诈骗（威慑力最强）

### 5.1 完整话术流程

| 步骤 | 话术内容 | 心理操控手法 |
|------|---------|-------------|
| ① | 「你好，我是XX市公安局民警，你名下的一张银行卡涉嫌洗钱案」 | 权威身份 + 突然袭击，制造恐慌 |
| ② | 准确说出受害者的姓名、身份证号、家庭住址（从黑产购买） | 信息准确 → 增加可信度 |
| ③ | 发送伪造的「逮捕令」「通缉令」（带红章、编号、受害者照片） | 视觉冲击力，强化恐惧 |
| ④ | 「案件涉密，不准告诉任何人，否则会连累家人」 | 隔离受害者，阻止其向亲友求助 |
| ⑤ | 诱导下载「安全防护」APP（实为远程控制/木马软件） | 技术控制 |
| ⑥ | 要求将资金转入「安全账户」进行「资金清查」 | 最终目的 |

### 5.2 2025 新变种：FaceTime + 伪造通缉令

- **手法**：通过苹果 FaceTime 视频通话冒充公检法，视频画面中骗子穿着假警服、背景是伪造的「公安局」办公室
- **结合技术**：使用改号软件将来电显示修改为当地公安局的真实号码
- **真实案例**：北京张女士接到 FaceTime 来电，对方显示为「北京市公安局」，视频中看到穿警服的「民警」，最终被骗 3 万元

## 六、虚假网络贷款诈骗

| 步骤 | 操作 |
|------|------|
| ① | 在搜索引擎/短视频投放「无抵押、低利息、秒放款」广告 |
| ② | 诱导下载仿冒银行/网贷 APP（界面与正规平台几乎一致） |
| ③ | 填写资料提交申请后，系统显示「审核通过，额度 XX 万」 |
| ④ | 点击提现 → 显示「银行卡号错误，资金被冻结」 |
| ⑤ | 客服要求缴纳「解冻费」（通常为贷款金额的 20%~30%） |
| ⑥ | 缴纳后 → 「还需刷流水验证还款能力」「保险费」「保证金」等连环收费 |
| ⑦ | 受害者多次缴费后仍无法放款，最终被拉黑 |

**关键点**：
- 诈骗 APP 的后台可以人工控制「错误提示」的内容和时机
- 所谓「银行卡号错误」实际上是骗子在后台手动修改受害者填写的卡号

**真实案例**：个体商户张先生因急需资金周转在网上申请贷款，对方以「账户异常需解冻」为由诱使其多次转账「验证资金」共计 12.3 万元。

## 七、注销校园贷/调整征信诈骗

| 步骤 | 操作 |
|------|------|
| ① | 冒充京东金融/支付宝/银监会工作人员来电 |
| ② | 话术：「你在学生时期注册过校园贷/京东白条，现在国家禁止校园贷，需要注销/关闭，否则影响个人征信」 |
| ③ | 诱导查询征信 → 受害者看到征信报告上的注册记录（实为正常的大学生信用卡/花呗记录） |
| ④ | 诱导从各个网贷平台（借呗、微粒贷、美团借钱等）借款 |
| ⑤ | 要求将借款转入「银监会清算账户」进行「清零操作」 |
| ⑥ | 承诺借款还清后可撤销贷款记录 → 实际款项已进入骗子账户 |

**核心逻辑**：利用受害者在大学期间可能开通过的小额信贷服务，制造「不注销就会影响征信」的恐慌。

**真实案例**：刚毕业的小刘接到「银监会工作人员」电话，称其名下学生贷记录影响征信需立即清理。恐慌之下从多个网贷平台借款 18 万元转入「清算账户」。

## 八、裸聊敲诈诈骗

| 步骤 | 操作 |
|------|------|
| ① | 在社交软件（陌陌/探探/ Soul）添加附近的人或匹配用户 |
| ② | 诱导下载指定「聊天 APP」（实为木马程序，安装后窃取通讯录） |
| ③ | 诱导视频裸聊，期间录制不雅视频 |
| ④ | 向受害者展示通讯录截图，以「向亲友群发视频」为要挟 |
| ⑤ | 索要「封口费」，金额通常从几百元起步，逐步升级到数万元 |

**技术细节**：

- 所谓「美女」多为提前录制的色情视频在摄像头前播放 + 实时语音互动
- 木马 APP 伪装成人脸识别/直播软件，安装时获取通讯录权限
- 敲诈金额采用「阶梯式」：第一次要 888→第二次 1888→第三次要求贷款转账

**真实案例**：公司职员吴先生深夜被「美女」搭讪，短暂视频裸聊后被发来不雅视频及通讯录截图，被迫转账 3.5 万元「封口费」。

## 九、虚假购物/游戏交易诈骗

| 类型 | 手法 |
|------|------|
| **游戏账号交易** | 在游戏内喊话低价出售高等级账号 → 诱导脱离官方平台（微信/QQ 私聊）→ 发送仿冒交易平台链接 → 以「解冻费」「保证金」收费 |
| **游戏装备/皮肤** | 伪装卖家在闲鱼/转转发布低价游戏道具 → 诱导点击钓鱼链接付款 → 收款后不发货拉黑 |
| **演唱会门票** | 在微博/小红书发布转让信息 → 诱导扫码付款 → 伪造购票截图发送给受害者 → 受害者到现场才发现票是假的 |

**真实案例**：中学生小赵欲购买「限量皮肤」，在非官方平台交易时，卖家称其「账户被冻结需充值解冻」，使用母亲手机多次转账共计 2.1 万元。

## 十、虚拟货币洗钱诈骗（2025 年最新）

| 步骤 | 操作 |
|------|------|
| ① | 冒充交易所客服/公检法，谎称账户存在洗钱风险 |
| ② | 诱导购买 USDT（泰达币）等稳定币 |
| ③ | 要求将 USDT 转入指定的「安全钱包地址」进行「资金验资」 |
| ④ | 承诺验资后原路退回 → 实际转入诈骗地址后立刻转走 |

**真实案例**：投资者郑先生接到「某交易所客服」电话，称其账户涉及洗钱需「配合调查」，按指引购买价值 25 万元的 USDT 转入「安全钱包」。
